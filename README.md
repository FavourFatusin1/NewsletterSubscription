# NewsletterSubscription
A real-time, verified email subscription system built using AWS Lambda, AWS SNS, AWS API Gateway, JavaScript, and Python.
This project enables users to subscribe with verified email addresses while ensuring scalability, serverless efficiency, and secure API interactions.
